******* Program Instructions *******
TO RUN THE PROGRAM
Step 1. Unzip the pa2 zipped file
Step 2. Place the pa2 folder on the desktop
Step 3. Open a terminal window and Navigate to the following file  ->  /desktop/pa2/pa2
Step 4. Type "make clean && make" , this will compile the file and remove residual files
Step 5. Run the program by typing "./kscore -k (#of clusters) -e (desired e value)"
	where (#of clusters) and (desired e value) are values provided by the user

TO EVALUATE THE PROGRAM
Step 6. Open an additional terminal window
Step 7. Navigate to /desktop/pa2
step 8. Type "python evaluate.py" . This will run the python scoring script and display 	the score.
